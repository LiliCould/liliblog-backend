## PUT 修改用户信息

PUT /api/auth/update

> Body 请求参数

```json
{
  "username": "string",
  "nickname": "string",
  "avatar": "string",
  "email": "string"
}
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|Authorization|header|string| 否 |none|
|body|body|[UserUpdateDTO](#schemauserupdatedto)| 否 |none|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "message": "string",
  "data": null,
  "timestamp": 0
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|[ResultVOVoid](#schemaresultvovoid)|
